Напишіть функцію get_and_zero(), яка якимось чином приймає на вхід 
(через параметр) цілочисленну змінну, в цю змінну заносить нуль,
а число що в ній було повертає в якості свого значення. 
Як потрібно цю функцію викликати?

int get_and_zero(int* num ) 
{
	int tmp = *num;
	*num = 0;
   	return tmp;
}

int num = 42;
int prevNum = get_and_zero(&num);
