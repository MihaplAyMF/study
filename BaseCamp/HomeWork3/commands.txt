    Основні команди для роботи з файлами та директоріями
ls - показує список файлів та директорій
cd - змінює поточну директорію
pwd - виводить шлях поточної директорії
cp - копіює файли чи каталоги
mv - переміщує або переймановує файли/каталоги
rm - видаляє файли чи каталоги
mkdir - створює нову директорію
rmdir - видаляє порожні директорії
touch - створє новий порожній файл або змінює час модифікації існуючого
ln - дозволяє створювати hard та symb посилання

Команди для роботи з файлами та текстом
cat - вивести вміст файлу
more, less - відобразити вміст посторінково
head - показує перші рядки файлу
tail - показує останні рядки файл
nano, vim - текстові редактори для роботи в терміналі
grep - шукати текст за вказаним шаблоном
find - шукати файли та каталоги за вказаними умовами
diff - порівнює файли та показує різницю між ними
echo - виводить текст або значення змінних на екран.

Команди для роботи з процесами
ps - показує список активних процесів
top, htop - моніторинг системи в реальному часі
kill - завершує процес за ідентифікатором PID
killall - завершує всі процеси з вказаною назвою
bg, fg - відправити процес у фоновий або передній план

Команди для роботи з правами доступу
chmod - змінює права доступу до файлів та каталогів
chown - змінює власника файлу або каталогу
sudo - магічне слово яке дозвоялє робити те що заманеться, навіть якщо ви пожалієте про це
su - позволяє змінити користувача
passwd - позволяє змінити пароль користувача
chsh - дозволяє змінити термінальну оболонку

Команди для роботи з архівацією
tar - створює та розпаковує архіви
zip, unzip - стискає та розпаковує файли у форматі zip
unrar - розпаковує файли у форматі rar
gzip, gunzip - стискає та розпаковує файли в форматі .gz

Команди для роботи з мережею
ping - перевіряє доступність до віддаленого вузла
ifconfig, ip - налаштовує мережеві інтерфейси
wget, curl - завантажує файли з мережі
ssh - підключається до віддалених серверів через захищене з'єднання
scp - копіює файли між комп'ютерами через SSH

Інформаційні команди
df - показує інформацію про використання диска
du - показує розмір файлів та каталогів
free - показує інформацію про систему
uname - відображає інформацію про систему
whoami - показує ім'я поточного користувача
date - показує або встановлює поточну дату та час
id - виводить інформацію про поточного користувача
groups - показує групи, до яких належить поточний користувач або вказаний користувач
env - виводить список змінних середовища, які визначені в поточному сеансі, з їхніми значеннями
dmesg - виводить повідомлення ядра про завантаження, апаратні пристрої, помилки та події
uptime - показує час роботи системи, кількість користувачів та середнє навантаження
lscpu - показує інформацію про процесор

Інші корисні команди
man -  показує довідку по команді
alias - створює псевдоніми для команд
history - показує історію виконаних команд
clear - очищує екран термінала
source - виконує вміст файлу у поточному сеані оболонки
set - виводить або задає змінні середовища та функції оболонки в поточному сеансі
eval - оцінює та виконує аргументи як команду оболонки
alias - створює псевдоніми для команд
